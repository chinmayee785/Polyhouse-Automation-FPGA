# 🌱 Smart Polyhouse Automation using FPGA

## 📌 Overview
This project automates polyhouse conditions using Verilog and FPGA.

## ⚙️ Tools
- Verilog HDL
- Xilinx Vivado

## 📁 Files
- polyhouse.v
- polyhouse.xdc

## 🧠 Working
System checks conditions and controls outputs like irrigation and temperature.
